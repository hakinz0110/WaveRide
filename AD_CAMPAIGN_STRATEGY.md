## 🎥 Video Ad Technical Specs

### Video Requirements:
- **Length:** 90 seconds ✅ (perfect for trading demo)
- **Format:** MP4 or MOV
- **Aspect Ratio:** 
  - Square (1:1) - 1080x1080 (BEST for feed)
  - Vertical (4:5) - 1080x1350 (good for mobile)
  - Landscape (16:9) - 1920x1080 (for desktop)
- **File Size:** Under 4GB
- **Resolution:** Minimum 1080p
- **Captions:** YES! Always add captions (85% watch without sound)

### Video Best Practices:
1. **Add text overlays** on screen (key points)
2. **First 3 seconds:** Show indicator working or bold statement
3. **Include captions** (Meta auto-generates, edit for accuracy)
4. **Brand logo:** Bottom corner throughout
5. **End screen:** "Get Yours - ₦10,800" + QR code (optional)
6. **Music:** Background music that doesn't overpower voice
7. **Thumbnail:** Choose eye-catching freeze frame (usually 0-3 sec mark)

### What to Show in 90 Seconds:
```
0-3s:   Hook - "Stop entering trades too late"
3-15s:  Problem - Show messy chart, late entries
15-30s: Solution - Introduce WaveRide indicator
30-60s: Demo - Show it working on live chart
60-80s: Benefits - List features with text overlay

```

## 📊 Key Metrics to Track

### In Meta Ads Manager:

**Performance Metrics:**
- **Amount Spent:** Should be ~₦7,000/day
- **Results (Purchases):** Track daily
- **Cost per Result:** Target ≤ ₦15,000 after Week 1
- **ROAS (Return on Ad Spend):** Target ≥ 1.5x
  - Formula: Revenue ÷ Ad Spend
  - Example: ₦64,800 ÷ ₦49,000 = 1.32x

**Delivery Metrics:**
- **Frequency:** Keep under 2.5 (how often same person sees ad)
- **CPM:** ₦2,000-5,000 (Cost per 1,000 impressions - LOWER for video)
- **CPC:** ₦30-100 (Cost per click - LOWER for video)
- **CTR:** Target ≥ 2% (Click-through rate - HIGHER for video)

**Video-Specific Metrics:**
- **ThruPlay:** Cost per 15-second view (₦8-15)
- **Video % Watched:** Average % of video watched (target 50%+)
- **3-Second Views:** Should be 80%+ of impressions
- **15-Second Views:** Should be 40%+ of 3-second views
- **Unique Viewers:** Total people who watched
- **Engagement Rate:** Reactions, comments, shares (target 3%+)

**Learning Phase:**
- Status should show "Learning" for 3-7 days
- Need 50 conversions per week to exit learning
- Once exited, performance stabilizes

### In Meta Events Manager:

**Pixel Health:**
- **PageView events:** Should fire constantly
- **Purchase events:** Should show "Server" badge (CAPI)
- **EMQ Score:** Target 6.0+ 
- **Event Match Quality:** Monitor daily

### In Selar Dashboard:

**Sales Data:**
- Track purchases against Meta attribution
- Compare timestamps
- Verify Meta is counting correctly

## 🆘 Troubleshooting

### Problem: No Purchases After 3 Days

**Check:**
- Is pixel firing? (Events Manager)
- Is CAPI connected? (Selar dashboard)
- Are people clicking? (Check CTR)
- Are people visiting page? (Check PageView events)
- Is payment link working? (Test yourself)

**Solutions:**
- If CTR < 0.5%: Change ad creative
- If CTR > 2% but no sales: Fix landing page
- If no PageView events: Fix pixel installation
- If PageView but no purchases: Check Selar CAPI

### Problem: Cost per Purchase Too High (>₦20,000)

**Wait 7 days first.** If still high after learning phase:

**Solutions:**
- Rewrite ad copy (different angle)
- Test video ad instead of image
- Narrow audience to best performers
- Add social proof to landing page
- Add urgency/scarcity to offer

### Problem: Low CTR (<0.5%)

**Solutions:**
- Use more eye-catching image
- Shorter headline (5 words max)
- Test shocking statement: "Stop Losing Money"
- Use numbers: "127 Traders Joined This Week"
- Test question headline: "Still Trading Blind?"
