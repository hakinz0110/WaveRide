# 🎬 Video Ad Launch Checklist

### Video File Ready:
- [ ] **Length:** 90 seconds ✅
- [ ] **Format:** MP4 or MOV
- [ ] **Resolution:** 1080p minimum
- [ ] **Aspect Ratio:** 1:1 (square - 1080x1080) RECOMMENDED
- [ ] **File Size:** Under 4GB
- [ ] **Audio:** Clear voiceover/narration

### Video Content Checklist:
- [ ] **0-3 seconds:** Strong hook (grabs attention)

- [ ] **45-75 seconds:** List benefits & features
- [ ] **75-90 seconds:** Clear CTA + price (₦10,800)
- [ ] **Captions:** Visible text overlays for key points
- [ ] **Branding:** Logo visible (bottom corner)
- [ ] **End Screen:** "Get Yours - ₦10,800" + URL

### Technical Quality:
- [ ] Good lighting (if showing face)
- [ ] Clear audio (no background noise)
- [ ] Smooth transitions
- [ ] Professional look (not shaky/blurry)
- [ ] Chart demo is clear and visible
- [ ] Text overlays are readable on mobile

## 🎬 Video Editing Tips (If You Need to Re-Edit)

### Hook Improvement (0-3 seconds):
- Use pattern interrupt: "Stop!" or "Warning!"
- Show shocking result/stat first
- Start with question: "Still trading blind?"
- Show indicator working immediately

### Middle Section (15-60 seconds):
- Keep showing chart examples (visual proof)
- Use text overlays to highlight key points
- Add "before/after" comparisons if possible
- Show actual trades (if you have permission)

### Ending (75-90 seconds):
- Make CTA clear and urgent
- Show price prominently: "₦10,800"
- Add scarcity: "Limited spots" or "Special price"
- End with logo + URL on screen

## 🆘 Troubleshooting Video Ads

### Problem: Video Not Approved
**Reasons:**
- Claims not supported (don't promise guaranteed profits)
- Before/after images (Meta doesn't allow for financial products)
- Misleading content
- Poor audio quality

**Solution:**
- Remove any "get rich" language
- Remove income claims/promises
- Focus on education and tool features
- Appeal if you believe it's compliant

### Problem: High Drop-off Rate
**If people leave within:**
- **0-3 seconds:** Hook is weak, change opening
- **15-30 seconds:** Too slow, get to demo faster
- **60-75 seconds:** Too long, cut to 60 seconds
- **Throughout:** Video is boring, add more energy

### Problem: High ThruPlay Cost (>₦25)
**Possible Causes:**
- Audience too narrow
- Video too long for audience
- Boring content
- Wrong placement

**Solutions:**
- Broaden audience
- Test 30-second cut of video
- Add more energy/excitement
- Try Feed + Reels only

### Problem: Good Views, No Clicks
**Means:** Video is engaging BUT not converting

**Solutions:**
- Make CTA stronger (end of video)
- Add urgency in text overlay
- Highlight price more prominently
- Test different primary text (above video)

## 💡 Pro Tips for Video Ads

### Boost Engagement:
1. **Add a question in first 3 seconds**
   - "Still entering trades too late?"
   - Makes people want to watch for answer

2. **Use text overlays generously**
   - 85% watch without sound
   - Highlight key benefits
   - Show price clearly

3. **Include social proof**
   - "200+ traders using this"
   - Flash testimonial screenshots
   - Show real results (if compliant)

4. **Create urgency at the end**
   - "Limited spots available"
   - "Special price ends soon"
   - "Join 200+ profitable traders"

### Lower Costs:
1. **Optimize for ThruPlay** instead of impressions
   - Meta charges per 15-sec view
   - Better engagement = lower cost

2. **Test different thumbnails**
   - Use most eye-catching frame
   - Add text overlay on thumbnail
   - A/B test 2-3 thumbnails

3. **Add captions manually**
   - Auto-captions have errors
   - Clean captions = better watch time
   - Better watch time = lower cost

4. **Keep people watching past 15 seconds**
   - That's the optimization goal
   - Hook must work in first 15 sec
   - Promise value delivery early
